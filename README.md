## 🚀 Featured Project

### 🧠 [TestCraft QA Agent](https://github.com/Anshul101010/TestCraft-QA-Agent)

> A Claude-powered Azure DevOps automation agent that converts Acceptance Criteria into structured, reviewable test cases and writes them directly to ADO.

| Feature | Detail |
|---|---|
| Coverage rules | Positive · Negative · Edge Case · Validation per AC point |
| Duplicate detection | Fetches recent module TCs before generating anything |
| Approval gate | Nothing writes to ADO without explicit user confirmation |
| TC formats | Web (Playwright-ready + Locator Registry) · Mobile (tap-based) |
| AC input sources | Paste · ADO User Story ID · Excel / PDF / Image upload |
| Confidence scoring | Coverage/5 and Executability/5 per TC |
